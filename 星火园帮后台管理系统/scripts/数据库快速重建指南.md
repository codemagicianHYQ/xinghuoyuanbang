# 星火园帮 - 数据库快速重建指南

> 适用于：腾讯云数据库因欠费被删除后，在新实例上重新建立完整库结构

## 一、准备工作

### 1. 获取新数据库连接信息

在新购买的腾讯云 MySQL 控制台获取：

- **主机地址**（host）
- **端口**（port，内网一般为 3306）
- **数据库名**（建议保持 `users`，与项目配置一致）
- **用户名**（一般为 `root`）
- **密码**

### 2. 修改项目配置

编辑 `config/database.js`，将连接信息改为新数据库：

```javascript
const sequelize = new Sequelize("users", "root", "你的新密码", {
  host: "你的新主机地址",  // 如 xxx.sql.tencentcdb.com
  port: 你的端口,          // 如 3306 或 24489
  dialect: "mysql",
  // ... 其余配置保持不变
});
```

## 二、执行 SQL 脚本（按顺序）

### 方式 A：腾讯云控制台执行

1. 登录腾讯云 MySQL 控制台 → 选择新实例 → **登录**
2. 使用 DMC 或命令行连接后，先**创建数据库**（若控制台未自动创建）：

   ```sql
   CREATE DATABASE IF NOT EXISTS users DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   USE users;
   ```

3. **第一步**：执行 `00_init_database_full.sql`  
   - 创建所有主表（communities、users、feedbacks、withdrawals、chats、resources、books、comments、follows、page_configs、exam_proofs、after_sales、user_earnings、system_messages、system_broadcasts、user_broadcast_reads、dispatchTeams、teamApplications、teamMembers、user_addresses）

4. **第二步**：执行 `01_create_first_community_and_shards.sql`  
   - 插入默认社区（id=1）  
   - 创建任务分表 `tasks_community_1`  
   - 创建校园互动分表 `campus_resources_community_1`  
   - 创建二手市集分表 `market_products_community_1`  

### 方式 B：命令行执行

```bash
cd 星火园帮后台管理系统

# 需要先安装 mysql 客户端，并将下方参数改为你的连接信息
mysql -h 你的主机 -P 端口 -u root -p users < scripts/00_init_database_full.sql
mysql -h 你的主机 -P 端口 -u root -p users < scripts/01_create_first_community_and_shards.sql
```

## 三、脚本说明

| 脚本 | 作用 |
|------|------|
| `00_init_database_full.sql` | 创建 20 张主表，包含用户、社区、反馈、提现、聊天、资料、图书、评论、关注、页面配置、考试凭证、售后、收益、系统消息、派单团队、用户地址等 |
| `01_create_first_community_and_shards.sql` | 插入首个社区、创建任务/校园互动/二手市集分表 |

两个脚本已包含项目当前所需的全部字段（含 `community_admin` 角色、`communityAdminId`、`refunded` 支付状态等），**无需再执行** `add_community_admin_field.sql`、`update_user_role_enum.sql` 等增量脚本。

## 四、验证

1. **检查表是否创建成功**：

   ```sql
   USE users;
   SHOW TABLES;
   ```

   应能看到 `communities`、`users`、`feedbacks` 等主表，以及 `tasks_community_1`、`campus_resources_community_1`、`market_products_community_1`。

2. **启动后端服务**：

   ```bash
   cd 星火园帮后台管理系统
   npm start
   ```

3. **创建超级管理员**：执行下方 SQL 插入管理员账号。

## 五、后续新增社区

通过后台管理系统的「社区管理」添加新社区时，系统会自动调用 `AutoShardingService` 创建对应的 `tasks_community_X`、`campus_resources_community_X`、`market_products_community_X` 分表，无需手动执行 SQL。

---

**提示**：如脚本中 `USE users;` 与你的数据库名不一致，请先修改脚本或创建同名数据库。
