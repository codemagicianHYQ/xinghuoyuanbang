# 社区管理功能使用指南

## 功能概述

社区管理模块提供了完整的社区信息管理功能，包括学校、小区等不同类型的社区管理。

## 主要功能

### 1. 社区列表管理

- ✅ 查看所有社区列表
- ✅ 支持分页显示
- ✅ 支持按类型、省份、城市筛选
- ✅ 支持关键词搜索
- ✅ 显示每个社区的用户数和任务数统计

### 2. 社区增删改查

- ✅ 添加新社区
- ✅ 编辑社区信息
- ✅ 删除社区（会检查关联数据）
- ✅ 查看社区详情

### 3. 批量操作

- ✅ 批量删除社区
- ✅ 批量更新社区类型

### 4. 数据统计

- ✅ 社区用户统计
- ✅ 社区任务统计
- ✅ 任务完成率统计
- ✅ 数据趋势图表

## API 接口

### 基础路径

```
/admin/api/v1/community-management
```

### 接口列表

#### 1. 获取社区列表

```
GET /communities
```

**参数：**

- `page`: 页码（默认 1）
- `limit`: 每页数量（默认 20）
- `search`: 搜索关键词
- `type`: 社区类型（school/community）
- `province`: 省份
- `city`: 城市

**响应示例：**

```json
{
  "success": true,
  "data": {
    "communities": [
      {
        "id": 1,
        "name": "华南理工大学",
        "type": "school",
        "province": "广东省",
        "city": "广州市",
        "district": "天河区",
        "address": "广东省广州市天河区五山路381号",
        "version": "campus",
        "userCount": 150,
        "taskCount": 25,
        "createdAt": "2025-10-11T01:40:56.000Z"
      }
    ],
    "total": 8,
    "currentPage": 1,
    "perPage": 20,
    "totalPages": 1
  }
}
```

#### 2. 创建社区

```
POST /communities
```

**请求体：**

```json
{
  "name": "新社区名称",
  "type": "school",
  "province": "广东省",
  "city": "广州市",
  "district": "天河区",
  "address": "详细地址",
  "version": "campus"
}
```

#### 3. 获取社区详情

```
GET /communities/{id}
```

#### 4. 更新社区

```
PUT /communities/{id}
```

#### 5. 删除社区

```
DELETE /communities/{id}
```

#### 6. 获取社区统计数据

```
GET /communities/{id}/stats
```

**参数：**

- `startDate`: 开始日期
- `endDate`: 结束日期

#### 7. 批量操作

```
POST /communities/batch
```

**请求体：**

```json
{
  "action": "delete",
  "communityIds": [1, 2, 3]
}
```

#### 8. 获取社区选项

```
GET /options
```

## 权限要求

所有社区管理功能都需要管理员权限：

- ✅ 需要有效的 JWT token
- ✅ 用户角色必须是管理员（admin）

## 数据模型

### Community 字段说明

- `id`: 社区 ID（主键）
- `name`: 社区名称（必填，唯一）
- `type`: 社区类型（school/community）
- `province`: 省份
- `city`: 城市
- `district`: 区县
- `address`: 详细地址
- `version`: 版本类型（campus/community）
- `createdAt`: 创建时间
- `updatedAt`: 更新时间

## 使用注意事项

1. **删除限制**：如果社区有关联的用户或任务，将无法删除
2. **名称唯一性**：社区名称必须唯一，不能重复
3. **必填字段**：社区名称、类型、省份、城市为必填字段
4. **权限控制**：只有管理员可以访问社区管理功能
